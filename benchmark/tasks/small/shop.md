# Shopping list

Build a shopping list: a text input plus Add button; each added item shows its
name, a quantity (starting at 1), a "+" button to increment the quantity, and a
remove button. Show the total number of items (sum of quantities) at the bottom.

Acceptance:
- Adding with an empty input does nothing.
- "+" increments only that row's quantity.
- Remove deletes the row; the total reflects quantities, not row count.
